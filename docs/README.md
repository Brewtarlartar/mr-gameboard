# The Tome

**The Tome** - Your comprehensive board game knowledge companion. A magical application that encompasses all board game information with an AI wizard chatbot, game library management, and interactive play mode guide.

## Features

- 🧙 AI Wizard Chatbot - Get game recommendations, rule clarifications, and strategy tips
- 🔮 Crystal Ball - Random game selection based on player count
- 📚 Game Library - Manage your collection with intelligent recommendations
- 🎮 Play Mode Companion - Step-by-step game setup and practice rounds
- 🎲 Game Utilities - Dice roller, score tracker, and turn timer

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, create a `.env.local` file with your API keys:

```
OPENAI_API_KEY=your_openai_api_key_here
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
